Instrucciones:

Con esta aplicación puedes crear y modificar tu propia lista de la compra para llevarla donde quieras, ¡todo desde tu dispositivo! Solamente necesitas tocar la pestaña "Cart" donde verás la lista de productos comprados y por comprar. Puedes acceder a cada una de estas categorias deslizando con el dedo para pasar de una a otra. Si rotas la pantalla del dispositivo, puedes ver todo el contenido haciendo scroll arriba/abajo para ver todo el contenido.
